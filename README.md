## 🔍 Search & Discovery Tips

- **Google Scholar**: Use the search operator `author:"First Last"` (e.g., `author:"Andrea Tagliasacchi"`) to find all publications. The "Search" links above are pre-filled queries guaranteed to surface the correct profile.
- **X (Twitter)**: Follow `#3DGS`, `#DynamicViewSynthesis`, and `#GaussianSplatting`.
- **GitHub**: Search `"dynamic-gaussian-splatting"` or `"4d-gaussian-splatting"`.
- **arXiv**: Subscribe to `cs.CV` and `cs.GR` for daily alerts.
